# 🐾 멍멍 스탑워치

귀여운 강아지 테마의 스탑워치입니다.

🔗 **라이브 데모:** [creativelotte-cmd.github.io/test](https://creativelotte-cmd.github.io/test/)

---

## 기능

- ▶ 시작 / 정지 / 계속
- 🐾 랩 타임 기록 (가장 빠른 랩 초록색, 가장 느린 랩 빨간색 표시)
- 초기화
- 타이머 동작 중 꼬리 흔들기 애니메이션
- 눈 깜빡임 애니메이션

## 사용법

`stopwatch.html` 파일을 브라우저에서 열거나 위 라이브 데모 링크에 접속합니다.
